# 📘 ERP 단어 연습장

ERP·물류·회계 용어를 학습하는 **단답식 퀴즈 애플리케이션**입니다.

## 🔗 주요 링크

- **배포 URL (바로 사용 가능)**
  👉 [https://erp-vocabulary-app.vercel.app](https://erp-vocabulary-app.vercel.app)

- **랜덤 퀴즈 모드**
  👉 약어만 보고 영문/한글을 입력해 학습
  [https://erp-vocabulary-app.vercel.app/quiz/random](https://erp-vocabulary-app.vercel.app/quiz/random)

- **리스트 선택 모드**
  👉 전체 용어 리스트에서 선택해 학습
  [https://erp-vocabulary-app.vercel.app/list](https://erp-vocabulary-app.vercel.app/list)

## ✨ 핵심 기능 요약

- **랜덤 퀴즈**: 약어 → 영문 → 한글 입력
- **리스트 선택 퀴즈**: 원하는 용어 선택해 학습
- **재도전 기능**: 같은 문제 다시 풀이
- **모바일 친화 UI**: 터치 중심, 단일 컬럼 구조
